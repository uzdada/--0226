# Dual Tire Drift Track (Made by San)

이 프로젝트는 **파이게임(Pygame)**을 사용한 원형 드리프트 게임입니다.  
자동차가 트랙에서 드리프트할 때 좌우 타이어 자국이 남고, 드리프트 점수를 얻을 수 있습니다.

---

## 실행 환경

- Python 3.12 이상 권장
- Windows, macOS, Linux 모두 가능
- 필수 라이브러리: `pygame`

---

## 설치 및 실행

1. **Python 설치**  
   [Python 공식 사이트](https://www.python.org/downloads/)에서 최신 버전을 설치합니다.  
   설치 시 **"Add Python to PATH"** 체크 필수.

2. **Pygame 설치**  
   터미널(명령 프롬프트)에서 다음 명령 실행:

   ```bash
   pip install pygame
